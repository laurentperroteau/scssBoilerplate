Installer SASS / Compass en 5min

  1. Installer Ruby pour Windows avec le Ruby Installer (la version recommandé) en cochant “Add Ruby executables to your Path”
  2. Le reste de l'instalation se passera dans le terminal
  3. ruby --help pour vérifier qu’il est bien installer
  4. gem install sass (attendre instalation)
  5. gem install compass (attendre instalation)
  6. Ouvrir le chemin vers le dossier qui contient cette page cd c:\...\NameProject\scss
  7. compass watch pour lancer la compilation
    1. Si nouveau projet: un fichier nameProject.cssi> est crée dans un dossier CSS
    2. Si projet déjà débuté: tout est prêt pour travailler dans les fichiers .scss
  8. ctrl + C pour arrêter compass

Différences avec LESS

  • Les fichiers qui commence par "underscore" ne sont pas compilé.
  • Le nesting est le même.
  • Les variables commence par $.
  • Les mixins commence par @.
  • Les sprites d'icône sont générés automatiquement (dans ce thème 2 icônes sont appelé avec un mixin, le normal et le rétina).
  • On peut utiliser des fonctions, ex :
    @function response($target, $container) {
         @return $target / $container * 100%;
    }
    width: response($widthContent, $widthMain);
  • On peux "extend" les classes (a utiliser avec précautions).

Deux commandes utiles

  • compass compile -e devopement --force pour passer en mode "dévelopement/production"
  • compass compile -c config.rb --force pour forcer le fichier config.rb si besoin
